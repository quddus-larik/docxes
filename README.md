# DocXes

A highly optimized documentation engine that support multiple ReactJS frameworks include (Remix, Astro.js, Next.js, Gatsby.js). It provide ease of integration with react frameworks.

### Features 
  1. Real-time document sync/hot reload
  2. Fuzzy Fuse.js searching
  3. Auto Generation Sidebar Content and navigations
  4. Centralized `x-meta.config.tsx` for UI customizations and configurations.
