### Introduction

Documentation <Highlight className="bg-green-200">Engine</Highlight> that is independent on frameworks build with Typescript focused on Scalability and Performance that reduce both Server and Client usage make documentation more efficient on build time. It build Pre-compiled documentation on build time that helps the web to fetch and load quickly from pre-compiled data generated by docxes engine.
